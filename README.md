# JGTool
开发中常用工具汇总

## 功能

- [x] NetWork  网络工具

        1、 AWSS3Manager  `AWSS3`亚马逊S3云存储封装
        2、 WebSocket  `SocketRocket`封装
        3、 HttpManager 网络请求封装
        4、 LocationUtil  定位封装
        5、 DownImageTool 下载图片封装
- [x] JGEmptyView 页面空视图 
- [x] JGToast Toast提示
- [x] StringAttribute 富文本简单使用
- [x] DEBUG 调试状态下的工具

        1、JGCreateModelFile 自动生成模型文件
        2、Log  打印中文 编码处理
        3、FPS 
- [x] Category 常用分类
- [x] CommonView 好用的自定义视图
